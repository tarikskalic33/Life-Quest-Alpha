# LifeQuest Platinum

A comprehensive goal tracking and life management app built with Flutter.

## Try it locally

```powershell
cd "c:\Users\wsk\Documents\LifeQuest Platinum\lifequest1"
flutter pub get
flutter run
```

## Run tests

```powershell
flutter test
```

## Project structure

- `lib/models/` — data models (e.g., `goal.dart`)
- `lib/providers/` — state management (e.g., `goals_provider.dart`)
- `lib/screens/` — UI screens
- `lib/widgets/` — reusable widgets
- `lib/services/` — (future) persistence and backend integration
